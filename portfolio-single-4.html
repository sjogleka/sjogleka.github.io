N-queens problem using Hill Climb

Project Details

  • Domain : Intelligent Systems
  • Completed On : Sept 2018 - Nov 2018

Project Link

Developed a python program to find solution to N-queen problem using hill climbing and its variant algorithms such as steepest ascent (with/without sideway moves), random restart etc.
Below is the algorithm that I followed while implementing the same.


• Algorithm for Hill Climbing:
1. Take number of queens and number of iterations from user,
2. Generate object of QueenState(),
3. Initialize all variables of QueenState with the help of default constructor,
4. Call to the specific algorithm will be made on the basis of user input(Lets consider user selects Steepest Ascent with side moves),
5. While True
6. All children of current state will get generated and will get store in children
7. Heuristic value(Number of queens attacking each other) of all generated nodes will get store in children_num_queen_attacks,
8. Minimum value of heuristic will get selected from all available heuristics which will be in best_child.
9. if (number of attacks in best_child > number of attacks in node):
10. break;
11. elseif (number of attacks in best_child == number of attacks in node):
12. if not allow_sideways or sideways_moves == max_sideways:
13. break;
14. else:
15. Increment sideways_moves
16. Select best node as child node
17. Now the result will have path, path length and outcome either ‘success’ or ‘failure’.
18. At last, print the path followed and average number of steps required.
